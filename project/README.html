My Favorite Albums (MFA)

My Favorite Albums is a specialized tool designed to analyze and visualize your annual music rankings. Whether you are a listener looking to explore your taste or a developer wanting to build custom data visualizations, MFA provides a modular framework for music data analysis.

🚀 Quick Start

  • Users: Learn how to navigate the interface and explore different sorting methods.
  • Developers: Download the source code, set up RStudio, and begin importing your own collection.
