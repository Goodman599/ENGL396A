Project Overview

My Favorite Albums is software that allows you to analyze annual rankings of select albums. For users, it includes many sorting and viewing methods to organize these albums.

By installing the software, developers can input their own favorite albums and analyze their selection.

Note:
The My Favorite Albums software does not support hosting your own version online. Instructions on web hosting are not covered in this documentation.

Key Definitions

  • Favorite Album: An album featured in the data provided to MFA.
  • Rating: A score from 1 to 10 inclusive.
  • Ranking: Determined by rating and separated by year in descending order (1 is the highest).

For Users

This section details the user interface and how the sorting methods work.

The Navigation Bar

The navigation bar at the top of the page contains links directing you to each sorting method.

Sorting Methods

  • Number One Albums: Displays the highest-rated album for a given year range.
  • Top Albums By Year: Shows every album released in a specific year, ranked.
  • Artists’ Albums: Lists all favorited albums by a specific artist with a statistics summary.
  • Favorite Artists: Finds your top 15 highest-rated artists based on custom criteria.
  • Artist Comparison: Plots two artists against each other in a line graph over time.
  • Vinyl: Suggests albums rated 7 or higher that you do not yet own on vinyl.
