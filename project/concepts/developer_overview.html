What is a .csv file?

MFA stores data in .csv format. It functions like a spreadsheet with these headers:
Year, Ranking, Album, Artist, Rating, Vinyl, EP, Live

Note:
There is an example version of My Favorite Albums, with an example CSV file and sorting methods.

The Tech Stack

  • R: The core programming language.
  • RStudio: The recommended IDE for editing code and data. This set of documentation will use RStudio as the IDE.

File System Rundown:

  • Sorting methods (i.e. all_albums.R): A file containing the code required to run a sorting method.
  • app.R: The "hub" for My Favorite Albums. Contains references to every sorting method. References are required to run functions.
  • app_ui.R: Defines all of the UI a sorting method needs. Defines all of the buttons, sliders, displays, etc., a function needs.
  • app_server.R: Defines how the functions of a sorting method are connected to the UI defined in app_ui.R. Without it, the UI would be purely decorative.
Note:
MFA is modular. You can experiment with new sorting methods without breaking the core software.
