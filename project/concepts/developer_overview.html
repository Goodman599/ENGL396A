Developer Concepts

MFA is built to be modular, allowing developers to personalize the experience by inputting data or adding logic.

Note:
MFA does not support online hosting. Instructions are for local environments only.

What is a .csv file?

MFA stores data in .csv format. It functions like a spreadsheet with these headers:

Year, Ranking, Album, Artist, Rating, Vinyl, EP, Live

The Tech Stack

  • R: The core programming language.
  • RStudio: The recommended IDE for editing code and data.
Note:
MFA is modular. You can experiment with new sorting methods without breaking the core software.
