Setting up My Favorite Albums

If you want to edit My Favorite Albums, such as importing personal album preferences or adding new data analyzing methods, you need to download My Favorite Albums.

Download R and RStudio

My favorite album runs on the programming language R, and the primary workspace for it is RStudio.
If you don't have RStudio installed yet, the official download is here.
From the RStudio page, download R too.

Warning:
Do not download the RStudio Server software

Download My Favorite Albums

To download My Favorite Albums, you need to:

  1. Open the official My Favorite Albums GitHub repository.
    • This is where all of the code for the example website is stored.
  2. Click the green Code button.
  3. Click Download ZIP.
    Note:
    The default download location for My Favorite Albums is your Downloads folder. Check there if you can't find it!
  4. Extract the file.

You should now be able to run any file in the My Favorite Albums folder using RStudio.

Set Up RStudio for My Favorite Albums

Warning:
If you have not installed R or RStudio yet, refer to the Downloading R and RStudio portion of the guide.

If this is your first time using RStudio, you're likely missing some key packages My Favorite Albums requires to work. Thankfully, downloading them is very straightforward:

  1. Open app.R using RStudio
  2. In the bottom left of RStudio, called the developer console, run this command: install.packages(c("shiny", "ggplot2", "dplyr))

Now, near the top center of RStudio, click Run App, and a copy of the example My Favorite Albums site should open.

If RStudio still prompts you to install or update any packages, follow the instructions shown in the pop-ups.
