Downloading My Favorite Albums

If you want to edit My Favorite Albums, such as importing personal song preferences or adding new data analyzing methods, you need to download My Favorite Albums.

Download R and RStudio

My favorite album runs on the programming language R, and the primary workspace for it is RStudio.
If you don't have RStudio installed yet, the official download is here.
From the RStudio page, download R too.

Warning:
Make sure to not download the RStudio Server

Download My Favorite Albums

To download My Favorite Albums, you need to:

  1. Open the official My Favorite Albums GitHub repository.
    • This is where all of the code for the example website is stored.
  2. Click the green Code button.
  3. Click Download ZIP.
  4. Extract the file.

You can now run any file in the folder using RStudio.

Set Up RStudio for My Favorite Albums

If this is your first time using RStudio, you're likely missing some key packages My Favorite Albums requires to work. Thankfully, downloading them is very straightforward:

  1. Open app.R using RStudio
  2. At the top of the left half of RStudio, there should be a yellow warning stating you are missing packages. Click Install located to the right of the warning text.
    • If not, at the top of the left half of RStudio, click Run App.
  3. In RStudio, open compare_bands.R, which requires another library. Repeat step 2.

Now, if you go back to app.R and click Rup App, a copy of the example My Favorite Albums site should open.
