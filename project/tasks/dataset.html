Creating and Adding a Custom .csv File

To import custom album data, you need to have both My Favorite Albums and RStudio installed and set up.
By creating a custom .csv file, you can change the album data My Favorite Albums can analyze.

Note:
This will not affect the example version of My Favorite Albums.

Creating a .csv File

There are two main methods to create a .csv file. The first is easier but requires external software, while the second is more tedius but built into R.

Using a Third Party Spreadsheet Software

Using a third party spreadsheat software, like Microsoft Excel, is the simplest way to create a .csv file for My Favorite Albums.
Alternatively, you could use an online csv editor, like the one on tableconvert.com.

To work with a spreadsheet software:

  1. Set the first row, or headers, of the spreadsheat to Year,Ranking,Album,Artist,Rating,Vinyl,EP,Live
  2. Fill out the spreadsheet normally for each of your favorite albums. Details are given in the Filling a .csv File section
  3. When you're done, download your spreadsheet as a .csv file.
    • If you are using excel or a similar software:
      1. Click File->Save As
      2. Select .csv as the file format.
    • If you are using tableconverter or a similar website:
      1. Download the file directly.
  4. Name the file album-rankings.csv

Using RStudio

RSudio comes with an in-built way to view and edit .csv files. While not requiring any external tools, the process can be more confusing and time consuming.

To work with RStudio:

  1. In the bottom right corner of RStudio, go to data
    • You can edit album-rankings.csv directly, or make a copy of it to preserve the original as an example.
  2. Fill the file out for each of your favorite songs. Details are given in the Filling a .csv File section
    • Treat each comma like a column in a spreadsheet, and each line change as a row. Commas with nothing between them are the same as an empty cell.
    • An example entry would be 1996,25,Becoming X,Sneaker Pimps,6,,,
  3. Make sure the file is named album-rankings.csv. If you made a copy, make sure the original is named something else.
