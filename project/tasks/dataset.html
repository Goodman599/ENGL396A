Creating and Adding a Custom .csv File

To import custom album data, you need to have both My Favorite Albums and RStudio installed and set up.
By creating a custom .csv file, you can change the album data My Favorite Albums can analyze.

This will not affect the example version of My Favorite Albums.

Creating a .csv File

There are two main methods to create a .csv file. The first is easier but requires external software, while the second is more tedius but built into R.

Using a Third Party Spreadsheet Software

Using a third party spreadsheat software, like Microsoft Excel, is the simplest way to create a .csv file for My Favorite Albums.
Alternatively, you could use an online csv editor, like the one on csvsuite.com.

To work with a spreadsheet software:

  1. Set the first row, or headers, of the spreadsheat to Year,Ranking,Album,Artist,Rating,Vinyl,EP,Live
    • If you are using Csvsuite, you can directly paste the headers into the Data Input box.
    • After that, you can scroll down to the Online Table Editor.
