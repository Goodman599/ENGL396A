Creating and Adding a Custom .csv File

Important:
If you don't know what a .csv file is, see the definition here.

To import custom album data, you need to have both My Favorite Albums and RStudio installed and set up.
By creating a custom .csv file, you can change the album data My Favorite Albums can analyze.

Important:
This will not affect the example version of My Favorite Albums.

Creating a .csv File

There are two main methods to create a .csv file. The first is easier but requires external software, while the second is more tedious but built into R.

Using a Third Party Spreadsheet Software

Using a third party spreadsheat software, like Microsoft Excel, is the simplest way to create a .csv file for My Favorite Albums.
Alternatively, you could use an online csv editor, like the one on tableconvert.com.

To work with a spreadsheet software:

  1. Set the first row, or headers, of the spreadsheat to Year,Ranking,Album,Artist,Rating,Vinyl,EP,Live
    . These define the information contained in each column of the spreadsheet.
  2. Fill out the spreadsheet normally for each of your favorite albums. Details are given in the Filling a .csv File section.
  3. When you're done, download your spreadsheet as a .csv file.
    • If you are using Excel or a similar software:
      1. Click File->Save As.
      2. Select .csv as the file format.
    • If you are using tableconverter or a similar website:
      1. Download the file directly.
  4. Name the file album-rankings.csv

Using RStudio

RSudio comes with a built-in way to view and edit .csv files. While not requiring any external tools, the process can be more confusing and time consuming.

To work with RStudio:

  1. In the bottom right corner of RStudio, go to data
    • You can edit album-rankings.csv directly, or make a copy of it to preserve the original as an example.
  2. Fill the file out for each of your favorite songs. Details are given in the Filling a .csv File section
    • Treat each comma like a column in a spreadsheet, and each line change as a row. Commas with nothing between them are the same as an empty cell.
    • Example:
      If the album Becoming X by Sneaker Pimps released in 1996 is a 7/10 and your 5th favorite album released in 1996, it can be entered as 1996,5,Becoming X,Sneaker Pimps,7,,,
      Important:
      This assumes you the album wasn't a live recording, the album is not an EP, and you don't own a vinyl for that album. The values to enter if any of these are false are shown in this table.
  3. Make sure the file is named album-rankings.csv. If you made a copy, make sure the original is named something else.

Filling Out Your Album Rankings File

This will be a rundown on how to fill each "column" of your album-rankings.csv file for it to work with My Favorite Albums.

Column Name Description Requirement / Example
Year The year the album was released. Written as a number (e.g., 1996)
Ranking Your rank for that album within that specific year. 1 is your top favorite.
Album The official title of the album. e.g., Becoming X
Artist The name of the performer or group. e.g., Sneaker Pimps
Rating Your personal score for the music. Scale of 1 to 10
Vinyl Indicates physical vinyl ownership. Enter v if you do, or leave blank otherwise.
EP Indicates if the release is an EP. Enter EP if it is, or leave blank otherwise.
Live Indicates if it is a live recording. Enter Live if it is, or leave blank otherwise.
Warning:
Be careful with rankings!
  • My Favorite Albums does not check if your ratings match your rankings. If you accidentally rank a 1/10 album as #1 in a year, My Favorite Albums will return erroneous data.
  • My Favorite Albums will not notify you if the rankings of any two albums in the same year are the same. If you rank two different albuyms released in 2003 both as #1 of the year, My Favorite Albums will return erroneous data.

Importing Your Album Rankings Into My Favorite Albums

If you made your rankings file using RStudio, you should already be done. If you didn't, or something didn't work:

  1. Open the local file location where you are storing My Favorite Albums
  2. Open the data folder
  3. Move your file into the data folder, replacing the old .csv file.
    • Ensure your file is called album-rankings.csv.
  4. Use RStudio to open app.R, and click Run App. Your data should now be analyzed by My Favorite Albums.
