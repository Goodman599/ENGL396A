Creating and Adding a Sorting Method

If you want to get more technical and customize how you view your music, you need to create your own sorting method. This guide is for all skill levels, so don't worry if this is your first time coding!
By using dplyr, you can create new ways to filter and rank your data.

Creating your Analysis Method

Most analysis methods in My Favorite Albums involve taking your main dataset and "piping" it through a series of functions. This section will be a rundown on some important functions you may use during the filtering process.
We will imagine the .csv file as a spreadsheet, with each row as an album and each column as a data entry.

Adding your analysis method to My Favorite Albums

Once your R code works correctly, you need to integrate it into the user interface so you can select it from the navigation menu.

  1. Open app.R in RStudio.
  2. Locate the server function where the data processing happens.
    • This is where you will paste the dplyr logic you created in the previous section.
  3. Update the ui (User Interface) section to add a new button or menu item.
  4. Click Run App to test your new sorting method in the browser.
Warning:
Ensure your column names in the code match the headers in your album-rankings.csv exactly (e.g., "Rating" with a capital R).

For more help with the code syntax, refer back to the dplyr crash course.
